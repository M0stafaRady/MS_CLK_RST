# MS_CLK_RST
All digital clock and rest controller

## Features
- PoR Generation (all digital)
- Internal RC Oscillator (8mhz, 16mhz, 32mhz, 64mhz and 128mhz)
- Clock Multiplexing
- Clock divider (/2, /4 and /8)
- External Clock Monitor (fail-safe)
